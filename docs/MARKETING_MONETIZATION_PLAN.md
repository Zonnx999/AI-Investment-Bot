## 1. 시장 기회 — 왜 지금인가

### 한국 개인 투자자 규모

| 지표 | 수치 | 출처 |
|---|---|---|
| 개인 주식 보유자 수 | **1,450만 명** (2025년 말) | KED Global |
| 2019년 대비 성장 | 600만 → 1,450만 (+141%) | CNBC |
| 활성 거래 계좌 수 | **1억 520만 개** (2026년 5월) | Al Jazeera |
| 개인의 거래량 비중 | **60–70%** (미국·일본은 30%) | Korea Times |
| 2025년 미국 주식 순매수 | **$736억** (전년 대비 5배) | CNBC |

**핵심 인사이트:** 한국 개인 투자자는 세계에서 가장 적극적인 집단 중 하나다. 전체 거래량의 3분의 2를 차지하고, 2025년 한 해에만 미국 주식을 $736억어치 순매수했다. "동학개미 → 서학개미"로 진화하면서 KR + US 이중 시장을 동시에 보는 수요가 폭발적으로 증가했다.

이 봇은 처음부터 KR + US + 크립토 세 시장을 동시에 다루도록 설계되어 있다. **이것 자체가 구조적 차별점이다.**

### 왜 지금 틈새가 있는가

한국 개인 투자자가 현재 쓸 수 있는 분석 도구는 크게 세 가지다:

1. **증권사 HTS 무료 스크리너** — 기본 PER/PBR 정렬, UI 구식, 퀀트 팩터 없음
2. **Seeking Alpha / InvestingPro** — 미국 주식 중심, 영어, 월 $25–$200. 한국 주식 커버리지 없음
3. **텔레그램/카카오 유료 채널** — 인간의 주관적 종목 추천, 월 ₩30,000–₩100,000, 신뢰도 불안정

**비어 있는 자리:** "한국어로, 한국+미국+크립토를 동시에, 결정론적 퀀트 신호로, 합리적인 가격에" 제공하는 서비스가 없다. 퀀터스(Quantus)가 가장 가까운 경쟁자이지만, 그쪽은 전략 백테스트·자동 매매(실행 레이어)에 집중하고 있고 2025년 12월 기준 7만 3천명 사용자에 운용자산 700억원 규모다. 즉 시장 자체는 이미 검증되어 있다.

## 2. 목표 고객 — 누가 돈을 낼 것인가

### 1차 타깃: 서학개미 (30–50대, 자기주도형)

- 국내 주식 + 미국 주식을 동시에 보유 (2025년 전체 개인 투자자의 상당수)
- 매일 아침 Naver Finance, 증권사 앱, 유튜브 채널을 순회하며 정보 수집
- "내가 직접 분석하고 싶은데 시간이 없다" — 인사이트는 원하지만 스프레드시트를 돌리기 싫다
- 텔레그램을 이미 쓴다 (암호화폐 커뮤니티 진입 경로로 많이 정착됨)
- **지불 의사 (WTP) 추정:** 증권사 HTS 부가서비스 (월 ₩5,500–₩11,000) 대비 2–3배 수준 ₩9,900–₩19,900에 익숙

### 2차 타깃: 재미·재일 한국인 투자자

- 미국 또는 일본 거주, 한국 주식도 동시 보유
- 한국어 서비스이지만 USD/JPY 결제 가능해야 함
- 한국어 퀀트 신호를 영어 서비스보다 훨씬 선호

### 배제할 고객

- "어떤 종목 사야 해요?"를 기대하는 완전 초보자 → 규제 리스크 + 민원 리스크 높음
- 알고리즘 자동 매매 원하는 유저 → 이 봇의 범위 밖 (Quantus, 키움 API 방향)

## 4. 수익화 모델 — 가격 근거

### 티어 구조

| 티어 | 가격 | 근거 |
|---|---|---|
| **무료** | ₩0 | 유입 채널. 실제 신호 맛보기 → 전환 유도 |
| **기본 (Basic)** | **₩9,900/월** | 심리적 ₩10,000 장벽 이하. 커피 2잔. 증권사 부가서비스 상한선과 동일 |
| **프로 (Pro)** | **₩19,900/월** | Netflix 기본(₩13,500) + 1.5배. 스트리밍처럼 "당연히 구독"하는 금액대 |
| **얼리버드** | **₩14,900/월 (평생 고정)** | 초기 50–100명 한정. 입소문 형성 인센티브 |
| **연간 할인** | 기본 ₩99,000/년, 프로 ₩199,000/년 | 2개월 무료 효과. 해지율 낮추는 가장 효과적 수단 |

### ₩9,900이 맞는 이유 (구체 근거)

1. **심리 앵커:** 한국 소비자는 ₩10,000 단위를 강하게 인식한다. ₩9,900은 "1만원 미만"으로 인지 → 충동 구매 가능 구간
2. **비교 벤치마크:** 키움증권 영웅문 프리미엄 부가서비스 ₩5,500–₩11,000/월. 이 봇은 그 범위 안에 들어감
3. **경쟁 대비 할인율:** Seeking Alpha Premium의 약 1/3 가격. "미국 서비스보다 저렴하고 한국 시장까지 커버" 메시지 작동
4. **LTV 계산:** 월 ₩9,900 × 12개월 × 유지율 60% = 연 ₩71,280/유저. 1,000명이면 연 ₩7,100만

### ₩19,900이 맞는 이유

1. **Netflix 기준점:** Netflix 기본 ₩13,500, 스탠다드 ₩17,000. "스트리밍 하나 끊고 퀀트 봇 구독"이라는 트레이드오프가 작동
2. **프리미엄 기능 정당화:** 워치리스트, 맞춤 알림은 기본 대비 실제 시간 절약이 크다
3. **가격 앵커링:** 프로 ₩19,900 옆에 기본 ₩9,900이 있으면 기본이 "합리적 선택"으로 보임 → 기본 전환율 올라감

### 수익 시나리오

| 단계 | 유료 구독자 | 평균 ARPU | 월 수익 | 연 수익 |
|---|---|---|---|---|

| 초기 (6개월) | 200명 | ₩13,000 | ₩2,600,000 | ₩31,200,000 |

## 5. 무료 티어 설계 — 전환 퍼널

무료 티어는 매출이 아니라 **신뢰 형성과 전환 준비**가 목적이다.

### 무료 티어에 줄 것

- `/start` → 즉시 활성화 (승인 불필요)
- 하루 1회 미국 시장 다이제스트 (상위 3종목 + 시장 국면)
- `/stock <ticker>` — 종목 점수 조회 (하루 3회 한도)
- `/help` — 기능 안내 + 업그레이드 안내

### 무료 티어에서 막을 것 (전환 유도 포인트)

- 한국 주식 다이제스트 (`/scan kr`) → "기본 플랜에서 이용 가능"
- 크립토 섹션
- 종목 조회 횟수 초과 시 → "오늘 한도를 다 쓰셨어요. /upgrade 로 무제한 이용"
- 워치리스트 `/watch`

### 전환 트리거 문구 예시

```
📊 오늘 /stock 한도(3회)를 모두 사용했습니다.

기본 플랜(₩9,900/월)으로 업그레이드하면:
• /stock 무제한
• 한국 + 미국 + 크립토 다이제스트
• 상위 10개 종목 스캔

👉 /upgrade
```

## 6. 고투마켓 전략 — 어떻게 알릴 것인가

### 채널 우선순위

#### 1순위: 커뮤니티 직접 접근 (비용 ₩0)

| 채널 | 방식 | 예상 효과 |
|---|---|---|
| **네이버 카페** (한국주식정보, 퀀트투자) | 직접 글 작성: "개인이 만든 퀀트 봇 베타 무료 공개" | 초기 100–300명 유입 |
| **클리앙 주식 게시판** | 사용 후기 형식으로 소개 | 50–150명 유입, 입소문 |
| **텔레그램 주식 채널** | 관련 채널 운영자에게 협업 제안 | 잠재적 수백 명 |
| **트위터/X 한국 투자 커뮤니티** | `#퀀트투자` `#서학개미` 해시태그 | 지속적 노출 |

#### 2순위: 콘텐츠 마케팅 (월 5–10시간)

- **네이버 블로그:** "레이 달리오 올웨더 전략을 텔레그램 봇으로 구현한 후기" 시리즈
- **GitHub:** README를 제품 소개 페이지처럼 다듬기 (스크린샷, 기능 GIF 포함)
- 직접 만든 봇의 **실제 추천 결과를 주간 스레드로** 공개 (투명성 = 신뢰)

#### 3순위: 유료 광고 (수익 발생 후)

- 네이버 검색 광고 키워드: "퀀트 투자 봇", "주식 신호 텔레그램"
- 예산: 초기 월 ₩200,000–₩500,000, ROAS 측정 후 확장

### 런칭 시퀀스 (베타 → 공개)

```
1주차: 봇 이름·소개 페이지 준비 + 얼리버드 50명 한정 선착순 공지
2주차: 네이버 카페·클리앙에 "베타 테스터 모집" 포스팅
3–4주차: 베타 피드백 수집, 치명적 버그 수정
5주차: 정식 오픈. 얼리버드 → 일반 구독으로 전환.
```

## 7. 법적 리스크와 대응

### 핵심 리스크: 투자자문업 해당 여부

한국 자본시장법상 "투자자문업"은 금융투자상품 가치 또는 투자판단에 관해 **개인별 조언**을 제공하고 대가를 받는 경우다. 이 봇이 주의해야 할 기준:

| 항목 | 이 봇의 현재 방식 | 위험도 |
|---|---|---|
| 개인화된 매매 추천 | ❌ 모든 구독자에게 동일한 시장 신호 | 낮음 |
| 특정 종목 "매수/매도" 지시 | ❌ "점수" + "근거" 제공 (판단은 사용자) | 낮음 |
| 포트폴리오 구성 조언 | ❌ 팩터 점수 조회만 | 낮음 |
| 대가 수취 | ✅ 구독료 수취 | 있음 |

**결론:** "점수 기반 스크리너 정보 제공 서비스"로 포지셔닝하면 투자자문업 해당 가능성이 낮다. 단, 아래를 반드시 갖춰야 한다:

1. 모든 메시지에 **"본 서비스는 투자 자문이 아닙니다"** 고지
2. `/start` 시 약관 동의 (클릭랩 또는 텍스트 응답)
3. 약관에 "개별 투자 결정은 사용자 본인의 책임" 명시
4. 워치리스트 기능도 "관심 종목의 공개 점수 조회"로 표현 — "이 종목 사세요"가 아님

> 법인 설립 후 "금융정보서비스업" 또는 "전자상거래업"으로 사업자 등록 권장 (개인사업자로도 시작 가능).

## 9. 브랜딩 — 이름과 정체성

### 왜 "AI-Investment-Bot"은 상용 이름으로 안 되는가

- 영어 + 기술 용어 → 일반 투자자에게 거리감
- "AI"를 앞세우면 "LLM이 추천하는 거 아냐?" 오해 유발 → 이 봇의 핵심 강점(결정론적)과 모순

### 이름 후보

| 이름 | 의미 | 느낌 |
|---|---|---|
