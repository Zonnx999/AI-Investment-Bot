# 마케팅 & 수익화 전략

> 이 문서는 AI-Investment-Bot을 1인 개인 툴에서 **팔 수 있는 SaaS 서비스**로 전환하기 위한
> 전략 문서입니다. 시장 데이터, 경쟁사 분석, 가격 논리, 단계별 실행 계획을 포함합니다.

## 1. 시장 기회 — 왜 지금인가

### 한국 개인 투자자 규모

| 지표 | 수치 | 출처 |
|---|---|---|
| 한국 개인 투자자 수 | **1,450만 명** (2025년 말 기준) | KED Global |
| 2019년 대비 성장 | 600만 명 → 1,450만 명 (+141%) | KED Global |
| 활성 주식 계좌 수 | **1억 524만 개** (2026년 5월) | Al Jazeera |
| 코스피 거래량 중 개인 비중 | **60–70%** (미국·일본은 약 30%) | Korea Times |
| 2025년 미국 주식 순매수 | **$736억** (전년 대비 5배 증가) | CNBC |

**핵심 인사이트 3가지:**

1. **규모가 크고 빠르게 성장 중.** 6년 만에 2.4배 증가. 이 추세는 금융문맹 개선, MTS 보급, 동학개미 운동으로 구조적으로 고착됨.

2. **국내외 동시 투자가 뉴노멀.** "서학개미"가 2025년 $736억을 미국 주식에 쏟아부었다. 이 봇이 KR+US+크립토를 동시에 커버하는 이유가 여기 있음. 경쟁사 대부분은 한국 장 또는 미국 장 중 하나만 커버.

3. **개인이 시장을 움직인다.** 거래량의 60–70%가 개인. 더 좋은 신호가 필요하지만, 기관급 툴에는 접근 불가.

### 갭 — 경쟁사가 못 채우는 자리

기존 선택지의 문제점:

| 서비스 | 문제 |
|---|---|
| 증권사 HTS (키움·미래에셋 등) | 무료지만 UI 구식, 팩터 분석 없음, 한국 장 한정 |
| Seeking Alpha Premium | $25/월. 미국 중심, 한국어 없음, 쿼트 신호 아닌 주관적 분석 |
| 퀀터스 (Quantus) | 전략 실행 플랫폼 (자동매매 중심), 73,000명 유저. 신호 분석보다 포트폴리오 실행에 치중 |
| 유료 텔레그램 채널 | ₩30,000–₩100,000/월. 사람이 쓰는 "핫픽" 중심, 결정론적 근거 없음 |
| FnGuide (한국 Bloomberg) | 기업용, 수백만원/년. 개인 접근 불가 |

**이 봇의 포지션:** 기관급 퀀트 신호 × 개인 가격 × 텔레그램 접근성. 이 조합은 공백이다.

## 2. 목표 고객 (페르소나)

### 주요 타깃: "데이터 기반 자기주도형 투자자"

- **연령대:** 30–50대
- **투자 성향:** 직접 종목 선택, 분산 투자 지향, ETF + 개별주 혼용
- **현재 고통:** 매일 수백 개 기사를 읽지만 신호가 명확하지 않음. HTS 차트는 볼 줄 알지만 PBR/ROE 기반 종목 스크리닝은 시간이 없음.
- **텔레그램 사용:** 암호화폐 투자자 커뮤니티를 중심으로 텔레그램 이미 익숙
- **지불 의사:** 커피 한 잔 값(주 1잔, 월 ₩10,000 이하)이면 시도 안 할 이유 없음

### 부차 타깃: "재미·재일 한국인 서학개미"

- KR + US 동시 계좌 보유
- 한국어로 KR 데이터를 읽고 싶지만 미국 시장 신호도 필요
- 이 봇이 두 시장을 한국어 다이제스트로 동시 커버하는 점이 독보적 차별점

### 잠재 규모 추산

```
한국 개인 투자자 1,450만 명
  × 10% (텔레그램 사용 + 퀀트 방법론에 관심)
  = 약 145만 명 잠재 도달 가능 시장 (SAM)

  × 1% (유료 전환 보수적 추산)
  = 약 14,500명 유료 구독자 가능

현실적 1년 목표: 500–2,000명 유료 구독자
```

## 4. 차별화 포지셔닝

### 핵심 메시지

> "매일 아침, 퀀트가 고른 종목 신호 — 한국·미국·크립토, 텔레그램으로"

### 경쟁사 대비 우위

| 차별점 | 이유 |
|---|---|
| **결정론적 신호** | AI 블랙박스가 아닌 레이 달리오·에드워드 소프 기반 수식. "왜 이 종목인가"를 설명할 수 있음 |
| **이중 시장 커버** | KR + US 동시. 서학개미 수요를 경쟁사 중 가장 잘 충족 |
| **텔레그램 네이티브** | 앱 설치 없음. 기존 텔레그램 투자 커뮤니티 그대로 활용 |
| **자동화** | 매일 아침 명령어 입력 없이 신호 수신. 패시브 인텔리전스 |
| **투명성** | `/stock AAPL` 요청 시 점수 근거(팩터별 점수)까지 보여줌. "그냥 믿어" 아님 |

### 피해야 할 포지셔닝

- "AI가 매수 타이밍을 맞춰드립니다" → 법적 위험 + 비현실적 기대
- "100% 수익 보장" → 법적 위험 + 신뢰 훼손
- "챗봇 투자 비서" → 이 봇의 본질이 아님. 신호 자동화 시스템임

## 5. 유통·마케팅 채널

### Phase 1: 커뮤니티 중심 (0–3개월, 예산 ₩0)

**타깃 커뮤니티:**

1. **네이버 카페 — 퀀트 투자 관련**
   - "퀀트 투자 연구소", "주식 가치 투자 카페" 등
   - 방법: 실제 분석 결과물(스크리닝 결과, 백테스트 차트)을 정보 글로 공유 → 봇 자연 언급
   - 절대 금지: 직접 광고 글 → 스팸 신고로 계정 정지

2. **텔레그램 투자 채널**
   - 기존 한국 주식/크립토 텔레그램 채널에서 활동하며 신뢰 구축
   - 방법: 데일리 다이제스트 일부를 무료 공개 채널에 24시간 지연 공개
   - 유료 채널은 실시간 + 풀 분석

3. **X (Twitter) 한국 금융 커뮤니티**
   - 한국 퀀트 트위터 (@퀀트투자, 금융경제 계정들)
   - 매일 스크리닝 상위 3개 종목 + 시장 국면 한 줄 요약 무료 공개
   - "전체 분석은 봇에서" 콜투액션

4. **클리앙 / PPOMPPU 주식 게시판**
   - 이미 "@kor_stock_bot"이 클리앙에서 언급된 선례 있음
   - "이런 봇을 만들었습니다 + 무료 체험" 형태의 자기소개 글

### Phase 2: 콘텐츠 마케팅 (3–6개월)

1. **GitHub Pages 랜딩 페이지**
   - `dashboard/landing.html`로 시작, 이후 별도 도메인 (예: `quantpick.kr`)
   - 기능 소개 + 가격표 + "무료로 시작하기" CTA (텔레그램 링크)

2. **주간 퀀트 레터 (무료)**
   - 매주 1회 공개 텔레그램 채널에 시장 국면 요약 + 이번 주 주목 섹터 공개
   - 유료 구독자 전환 퍼널로 활용

3. **YouTube 콜라보 (6개월+)**
   - 한국 금융 유튜버(구독 10만–50만 수준)와 협업
   - "이 봇이 어떻게 작동하나" 실제 시연 영상
   - 유튜버 전용 쿠폰 코드 → 유입 추적 가능

### Phase 3: 유료 획득 (6개월+, 수익 발생 후)

- 네이버 성과형 광고 (CPC) — 타깃 키워드: "퀀트 투자", "주식 스크리닝", "가치주 발굴"
- 텔레그램 채널 스폰서십

## 6. 법적 고려사항 (반드시 해결할 것)

### 투자자문업 이슈

한국 자본시장법상 **"투자자문업"** 은 개별화된 투자 조언을 유상으로 제공하는 경우 금융위원회 등록이 필요합니다. 핵심 구분:

| 허용 (등록 불필요) | 주의 필요 | 금지 (등록 필요) |
|---|---|---|
| 공개 스크리너 데이터 제공 | 개인 워치리스트 기반 "당신에게 맞는 신호" | 개별 종목 직접 매수 추천 + 수수료 |
| 동일 신호를 전 구독자에게 배포 | 포트폴리오 구성 조언 | 손익 보장 |
| 통계·팩터 기반 스코어 공개 | | 매매 시점 특정 ("지금 매수") |

**이 봇의 포지션:** 모든 신호는 동일하게 전체 구독자에게 배포되는 "금융정보 서비스"로 설계. 개인화된 투자 추천이 아님.

**필수 조치:**
1. 모든 메시지에 고지 문구 삽입: "본 서비스는 투자 자문이 아닙니다. 모든 투자 결정은 본인의 판단과 책임 하에 이루어져야 합니다."
2. `/start` 플로우에 약관 동의 단계 추가
3. 서비스 약관에 명시: "퀀트 알고리즘 기반 통계 정보 제공 서비스이며 투자 결과를 보장하지 않음"
4. 개인 워치리스트 기능은 "관심 종목의 공개 스코어 조회" 로 표현 (조언 X)

### 결제 사업자 등록

Toss Payments 또는 NHN KCP 사용 시:
- 통신판매업 신고 필요 (자치단체 신고, 비교적 간단)
- 사업자등록증 필요 (개인사업자 가능)
- 부가가치세: 유료 구독 매출에 10% 부가세 발생 (매출 포함 가격으로 설정하거나 별도 고지)

## 8. 브랜딩 제안

현재 이름 "AI-Investment-Bot"은 내부 식별자. 상용화 브랜드는 별도로 필요.

**후보 브랜드명 (참고용):**

| 이름 | 특징 |
|---|---|
| 퀀트픽 (QuantPick) | 직관적. 퀀트 + 종목 픽. 도메인 확인 필요 |
| 알파시그널 (AlphaSignal) | 영문 믹스. 글로벌 감각 |
| 앤트봇 (AntBot) | 동학개미·서학개미 레퍼런스. 친근함 |
| 달리오봇 | 철학 직결. 퀀트 투자자에게 즉각적 신뢰 |

**브랜딩 필수 요소:**
- 텔레그램 봇 이름 (`@your_brand_bot`) — BotFather에서 변경 가능
- 봇 설명 (4글자 이내 + 한 줄 소개)
- 봇 아바타 (로고 이미지)
- 랜딩 페이지 도메인 (`.kr` 또는 `.io`)

## 9. 위험 요소 및 대응

| 위험 | 가능성 | 대응 |
|---|---|---|
| 투자자문업 규제 적발 | 중 | 개인화 추천 금지, 약관 명시, "금융정보 서비스"로 포지션 |
| FMP API 비용 급증 (유저 증가 시) | 중 | `/stock` 결과 7일 캐시로 히트율 높음. 유저 1,000명 수준까지는 현재 플랜으로 커버 가능 예상 |
| 텔레그램 의존 단일 채널 | 중 | 랜딩 페이지에서 이메일 수집. 장기적 카카오봇 또는 웹 알림 검토 |
| Oracle 서버 OOM (트래픽 증가) | 중 | Toss 웹훅은 Cloudflare Worker로 분리. 봇 폴링만 Oracle에 |
| 경쟁사 복사 (퀀터스 등) | 낮음 | 선점 + 커뮤니티 신뢰가 해자. 기술 자체보다 데이터 파이프라인 + 브랜드 |
| 구독자 이탈 (첫 2–3개월) | 높음 | 위클리 요약 메시지, 워치리스트로 재참여 유도 |

## 10. 실행 우선순위 요약

```
1주차: 셀프서비스 가입 + /upgrade 스텁 + 고지 문구 → 베타 모집 시작
2주차: 랜딩 페이지 오픈 + 커뮤니티 글 시작 → 첫 50명 모집
3주차: 결제 연동 착수 + 티어 분기 구현
4주차: 베타 피드백 반영 + 얼리버드 종료 → 정식 론칭
```

목표: **런칭 후 3개월 이내 유료 구독자 100명**, 6개월 이내 500명.
